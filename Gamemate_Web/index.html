Commands:

  • Admin:
  • ban - Ban a user from the server
  • clear - Clear a number of messages in a channel
  • kick - Kick a user from the server
  • mute - Mute a user in the server
  • unban - Unban a previously banned user
  • warn - Warn a user
  • Debug:
  • debug_ping - Pings Client
  • Fun:
  • magic8ball - Magic 8 Ball
  • roll - Roll Dice
  • Minecraft:
  • mcserverinfo - Get server info
  • mcwiki - Minecraft Wiki
  • ranmcwiki - Random Minecraft Wiki page
  • Roblox:
  • rouseravatar - Fetches a Roblox user's avatar
  • rouserinfo - Fetches Roblox user info
